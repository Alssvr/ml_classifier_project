# Классификатор технических наименований

Проект для автоматической классификации технических наименований деталей и оборудования на основе TF-IDF и логистической регрессии с поддержкой активного обучения.

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd text_classifier_project

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Установить зависимости
pip install -r requirements.txt
