Профессиональный классификатор для автоматической категоризации технических наименований деталей и оборудования с поддержкой активного обучения и экспертных циклов валидации.

## 🎯 Возможности

- **Высокоточная классификация текстов** на основе продвинутого TF-IDF (30,000 признаков, n-граммы 1-3)
- **Логистическая регрессия** с балансировкой классов и оптимизированной регуляризацией
- **Активное обучение** через Cleanlab для автоматического обнаружения "шумных" меток
- **Интерактивный эксперт-цикл** для верификации сомнительных случаев
- **Пакетная обработка** новых наименований через bat-скрипты
- **Аугментация редких классов** для борьбы с дисбалансом данных

## 📊 Технические характеристики

### Параметры векторизации (TF-IDF)
```python
{
    "max_features": 30000,      # максимальный размер словаря
    "ngram_range": (1, 3),       # униграммы + биграммы + триграммы
    "min_df": 1,                 # учитываем даже редкие термины
    "max_df": 0.8,              # игнорируем слишком частотные (80% документов)
    "sublinear_tf": True,       # сглаживание TF (1 + log(tf))
    "norm": "l2",               # L2-нормализация векторов
    "use_idf": True,            # взвешивание по IDF
    "smooth_idf": True          # сглаживание IDF (1+log)
}

## Параметры классификатора (Logistic Regression)

{
    "classifier_type": "logistic_regression",
    "C": 1.0,                   # обратная сила регуляризации
    "max_iter": 1000,           # максимум итераций оптимизации
    "class_weight": "balanced", # автоматическая балансировка классов
    "solver": "lbfgs",         # эффективный оптимизатор для L2
    "random_state": 42,         # воспроизводимость результатов
    "n_jobs": -1               # использование всех ядер CPU
}
