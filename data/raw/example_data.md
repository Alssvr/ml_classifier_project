# Пример структуры данных

## classified_40k.xlsx
Размеченные данные для обучения (40 000 записей)

Колонки:
- ID: уникальный идентификатор
- Наименование: текст для классификации
- Шаблон класса: целевая категория

Пример:
| ID | Наименование | Шаблон класса |
|----|--------------|---------------|
| БП-00060124 | Винт Y60-140 3*13*19 | Винты гребные |
| БП-00011065 | Фланец 1-40-16,ГОСТ 12820-80 | Фланцы трубопроводные |

## unclassified_18k.xlsx
Данные для классификации (18 000 записей)

Колонки:
- ID: уникальный идентификатор
- Наименование: текст для классификации
